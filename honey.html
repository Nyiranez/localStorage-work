WEB STORAGE

This example is designed to demonstrate usage of the W3C Web Storage API. It should work as far back as IE8.
Choose custom background colours, logos and fonts from the below drop down menus, then try closing and
reopening the page — you will find that your choices are remembered, as they are stored using Web Storage. You
can also visit the storage event output (opens in new tab). Open this, change some values in the index page, then
look at the events page — you'll see the storage changes reported
